Architecture in motion

Span fingerprint matched

Notes become claims.

Memora reads markdown notes, extracts atomic claims with source byte spans, and preserves provenance to the exact location where each statement originated.

Claims get fingerprinted.

Every extracted source span gets a cryptographic fingerprint. At response time, cited claims are re-read from markdown and re-hashed before they are trusted.

Citations are enforced.

If the model cites a missing or mismatched claim, Memora rejects it and retries using verified-only context so unsupported citations do not pass silently.

Most personal AI memory tools trust the LLM. Memora doesn't have to.
